# Image-Browser

When a book is split in images, this page can browse by buttons and slider

# Adaptation

This program is for 44 images.

To change the number of images you must change the max of the slider and the value of last.

The names of the images are s_01.png to s_99.png

You can change prefix or extention in lines :

`image.setAttribute("src", "s_"+affP.innerHTML+".png");`
